### AI 개요 AI 는 기계 혹은 시스템 상 에서 만들 어 지 는 지능 으로서 인간 의 지능 과 유사 하 ᆫ 지 적 능력 을 인공 적 으로 구현 하 ᆫ 것 이 다 . - ( 머신러닝 ) AI 방식 중 하나 로서 인간 의 명령 이 아니 ᆫ 기계 스스로 데이터 로부터 학습 하 어 찾아내 ᆫ 패턴 에 따르 어 작업 을 실행 하 는 알고리즘 이 다 . - ( 딥러닝 ) 머신러닝 방식 중 하나 로서 뇌신경 망 을 모방 하 ᆫ 인공신경망 을 통하 어 데이터 를 학습 하 ᆫ AI 모델 이 다 . ### AI 의 분류 | 인공지능 | 머신러닝 | 딥러닝 | | --- | --- | --- | | 지 적 능력 을 인공 적 으로 구현 | 학습 을 통하 어 데이터 로부터 패턴 을 찾 어 지 적 능력 을 구현 | 뇌 를 모방 하 ᆫ 신경망 을 구성 하 고 데이터 를 학습 하 어 지 적 능력 을 구현 | | 전문가 시스템 | 결정 트리 | 합성곱 신경망 | | 규칙 기반 시스템 | 선 형 회귀 | 심층 강화학습 | | 사이버네틱스 | 퍼셉트론 | 순환 신경망 | ### AI 의 학습방식 AI 의 학습방식 으로서 는 지도 학습 , 비지 도 학습 , 강화학습 으로 분류 하 ᆯ 수 있 다 . - ( 지도 학습 ) 정답 이 주어지 ᆫ 상태 에서 AI 가 학습 하 는 방식 이 다 . - ( 비 지도 학습 ) 정답 을 알리 어 주 지 않 은 상태 에서 AI 가 스스로 데이터 의 특성 을 학습 하 고 패턴 을 파악 하 는 방식 이 다 . - ( 강화학습 ) 정해진 환경 안 에서 동작 하 는 에이전트 가 현재 의 상태 를 인식 하 고 , 선택 하 ᆯ 수 있 는 행동 중 보상 을 최대 화 하 는 행동 을 선택 하 어 학습 하 는 방식 이 다 . - 과거 에 도 존재 하 던 학습방식 이 었 으나 딥러닝 과 알파고 등장 이후 더 크 게 주목
