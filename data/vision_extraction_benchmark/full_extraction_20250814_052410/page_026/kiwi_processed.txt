## 제3 장 AI 학습데이터 및 모델 보안관 리본 장에서는 보안 관점에서 AI 모델의 개발주기 별로 AI 학습데이터 및 모델을 안전하게 관리하는 방안에 대해 안내한다.

- AI 모델개발 주기는 AI 모델개발을 위해 데이터수집부터 전처리· 설계· 학습· 검증· 테스트를 수행하는 과정이다.
- 금융회사 등은 자체적으로 AI 모델개발 주기 별로 보안 점검 항목을 개발하고 자체 점검을 수행하여 AI 보안성을 높일 수 있다.

[AI 모델개발 주기 단계]

| 단계 | 설명 |
|------|------|
| 1    | AI 학습데이터 수집 |
|      | - AI 모델학습에 필요한 데이터수집 단계 |
|      | - 사내 내부 또는 외부 공개된 소스를 통한 데이터수집 |
| 2    | 학습 데이터전처리 |
|      | - AI 모델에 적합한 형태로 변환하고 데이터 오염 여부를 확인하는 단계 |
|      | - 클리닝, 통합, 변환, 축소 등의 작업 수행 |
| 3    | 모델 설계· 학습 |
|      | - AI 모델을 설계하고 학습하는 단계 |
|      | - 강건성 높도록 설계하는 것이 중요 |
| 4    | 모델검증· 테스트 |
|      | - AI 모델의 성능 확인 및 테스트 단계 |
|      | - 모델 입력, 출력 값, 정보가 유출되지 않는 것이 중요 |
