### 제 3 장 . AI 학습데이터 및 모델 보안 관리 #### 4 AI 모델검증 · 평가 ** 가 어 . 개요 ** - 학습 후 AI 모델 의 성능 을 확인 하 기 위하 어서 검증 및 평가 ( 테스트 ) 를 시행 하 는 단계 이 다 . - ( 모델검증 ) 학습 이 완료 되 ᆫ 모델 을 검증 하 기 위하 ᆫ 단계 로 학습 시 에 이용 되 ᆫ 데이터 셋 의 일부 를 검증 에 이용 하 ᆫ다 . - 홀드 아웃 검증 , K 폴드교체 검증 등 의 기법 활용 가능 - ( 평가 ) 학습 과 검증 후 에 모델 의 최종 성능 을 평가 하 기 위하 어 테스트 하 는 단계 로 학습 · 검증 용 데이터 셋 과 별개 의 평가 데이터 셋 을 준비 하 어 평가 에 만 사용 하 ᆫ다 . ** 참고 ** AI 서비스 개발 시 의 데이터 셋 구분 < sup > 10 ) < / sup > ### 전체 데이터 셋 | 학습 | 검증 | 테스트 | | ------ | ------ | -------- | | 학습 셋 | | 평가 셋 | ** 나 . 보안 고려 사항 ** 1.
