## 금융분야 AI 보안 가이드라인 ### 2. 학습데이터 전 처리 #### 가. 개요 - 학습 데이터전처리 는 수집 하 ᆫ 데이터 를 AI 모델 입력 이나 분석 에 적합 하 ᆫ 형태 로 가공 하 는 단계 이 다 . - 전처리 과정 을 통하 어 학습데이터 의 품질 뿐 만 아니 라 AI 모델 의 보안 성 개선 도 가능 하 다 . - 데이터 의 품질 및 보안 성 은 AI 모델 의 성능 과 도 직접 적 으로 관련 이 있 으며 , 학습데이터 전 처리 단계 에서 데이터 의 품질 및 보안 성 을 확보 하 는 것 이 중요 하 다 . #### 나. 보안 고려 사항 1. ( 이상 치 처리 ) 데이터 내 에 불 필요 하 ᆫ 값 을 제거 하 고 이상 하 지 를 검출 하 어 처리 하 ᆫ다 . - 이 상치 처리 는 방식 에 따르 어서 기계 적 전처리 와 의미 적 전처리 로 분류 하 ᆯ 수 있 다 . - ( 기계 적 전처리 ) 데이터 의 통계 치 를 기반 으로 데이터 를 변환 하 거나 데이터증강 등 의 기법 을 통하 어서 데이터 품질 을 확보 하 ᆫ다 . - ( 의미 적 전처리 ) 데이터 에 대하 ᆫ 도메인 지식 을 바탕 으로 변환 하 어 데이터 의 품질 을 확보 하 ᆫ다 . - 클러스터링 , 레이블 링 , 특징 공학 등
