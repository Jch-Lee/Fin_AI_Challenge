## AI 모델 평가 지표 | 평가 지표 | 설명 | | --- | --- | | 정확도 ( Accuracy ) | 예측 하 ᆫ 데이터 셋 중 에서 정확 하 게 예측 하 ᆫ 비율 | | 오차 행렬 ( Confusion Matrix ) | 예측 값 과 실제 값 사이 에 탐지 값 을 측정 하 ᆫ 표 | | 정밀도 ( Precision ) | True 예측 결과 에서 실제 True 를 탐지 하 ᆫ 비율 | | 재현 율 ( Recall ) | 실제 True 값 에서 True 를 탐지 하 ᆫ 비율 | | F - 점수 ( F - score ) | 정밀도 와 재현 율 의 조화 평균 | | AUC - ROC 곡선 ( AUC - ROC Curve ) | 재현 율 과 특이 도 를 기준 으로 그리 어 지 는 곡선 | | 평균 절대 오차 ( MAE ) | 정답 과 예측 간 의 절대 거리 를 평균 하 ᆫ 거리 | ### 정확도 ( Accuracy ) - 예측 하 ᆫ 데이터 셋 중 에서 정확 하 게 예측 하 ᆫ 비율 ### 오차 행렬 ( Confusion Matrix ) - 예측 값 과 실제 값 사이 에 탐지 값 을 측정 하 ᆫ 표 - ** 실제 값 ** - ** 긍정 ( Positive ) ** - 참 양성 ( TP ) ( True Positive ) - 거짓 양성 ( FP ) ( False Positive ) - ** 부정 ( Negative ) ** - 참 음 성 ( TN ) ( True Negative ) - 거짓 음성 ( FN ) ( False Negative ) ### 정밀도 ( Precision ) - True 예측 결과 에서 실제 True 를 탐지 하 ᆫ 비율 - TP / ( TP + FP ) ### 재현 율 ( Recall ) - 실제 True 값 에서 True 를 탐지 하 ᆫ 비율 - TP / ( TP + FN ) ### F - 점수 ( F - score ) - 정밀도 와 재현 율 의 조화 평균 - 2 × 정밀도 × 재현 율 / ( 정밀도 + 재현 율 ) ### AUC - ROC 곡선 ( AUC - ROC Curve ) - 재현 율 과 특이 도 를 기준 으로 그리 어 지 는 곡선 - TN / ( TN + FP ) - AUC 수치 가 높 을수록 모델 의 성능 이 좋 음 ### 평균 절대 오차 ( MAE ) - 정답 과 예측 간 의 절대 거리 를 평균 하 ᆫ 거리 - 평균 절대 오차 수치 가 작 을수록 모델 의 성능 이 좋 음 --- ## AUC - ROC 곡선 ( AUC - ROC Curve ) - 재현 율 을 Y 축 으로 , 민감 도 를 X 축 으로 그리 는 곡선 - AUC = 0.5 : 랜덤 모델 , AUC = 0.7 : 중간 수준 모델 , AUC = 1 : 완벽 하 ᆫ 모델
