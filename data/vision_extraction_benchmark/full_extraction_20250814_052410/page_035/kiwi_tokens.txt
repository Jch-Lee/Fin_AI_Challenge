AI 학습데이터 및 모델 보안 관리 ## AI 모델 의 구조 - AI 모델 은 1 개 의 입력 층 ( Input Layer ) 과 출력 층 ( Output Layer ) , 1 개 이상 의 은닉 층 ( Hidden Layer ) 으로 이루어지 어 있 음 - AI 모델 의 층 은 노드 로 이루어지 어 있 으며 , 각 노 들 는 가중 치 를 보유 하 며 , 다른 노드 와 엣지 로 연결 되 어 전달 받 은 정보 를 엣지 를 통하 어 전달 - 위 AI 모델 은 ① 3 개 의 노드 로 이루어지 ᆫ 입력 층 , ② 2 개 의 노드 를 보유 하 ᆫ 3 개 의 은닉 층 , ③ 3 개 의 노드 로 이루어지 ᆫ 출력 층 으로 구성 되 어 있 음 ### 매개변수 축소 - ( 매개변수 축소 ) AI 모델 의 성능 을 유지 하 ᆫ 채 , 구조 를 변경 하 는 기법 - 해당 기법 을 통하 어 공격자 가 변형 되 ᆫ 모델 을 파악 하 기 어렵 게 만들 ᆯ 수 있 음 ## 파라미터 축소 기법 개요 - [ 기준 모델 ] - 3 개 의 노드 로 이루어지 ᆫ 입력 층 - 2 개 의 노드 를 보유 하 ᆫ 3 개 의 은닉 층 - 3 개 의 노드 로 이루어지 ᆫ 출력 층 - [ 매개변수 축소 모델 ] - 3 개 의 노드 로 이루어지 ᆫ 입력 층 - 2 개 의 노드 를 보유 하 ᆫ 2 개 의 은닉 층 - 3 개 의 노드 로 이루어지 ᆫ 출력 층 ## 회피공격 방식 - 회피공격 은 AI 모델 의 경사도 와 경계선 을 찾 어 , AI 모델 의 결정 을 회피 하 는 방식 을 사용 하 기 때문 에 경사도 를 마스 킹 하 어 노출 을 방지 하 면 공격 을 어렵 게 하 ᆯ 수 있 다 . - 또는 경사도 자체 를 정규 화 하 어 경사도 에 대하 ᆫ 정보 를 공격자 가 유추 하 기 어렵 게 하 ᆫ다 . ### 참고 문헌 7 ) Anish Athalye et al. , “ Obfuscated Gradients Give a False Sense of Security : Circumventing Defenses to Adversarial Examples ” , 2018. 8 ) Papernot , Nicolas , et al. , “ Distillation as a Defense to Adversarial Perturbations Against Deep Neural Networks ” , 2016. ``` --- 이 내용 은 AI 모델 의 구조 와 매개변수 축소 기법 에 대하 ᆫ 설명 을 포함 하 고 있 습니다 . 또한 , 회피공격 방식 에 대하 ᆫ 설명 과 함께 , 이 에 대하 ᆫ 연구 논문 의 예시 도 제시 되 었 습니다 .
